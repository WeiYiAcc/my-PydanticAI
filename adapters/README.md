# my-PydanticAI adapters

Planned adapter groups:
- `workers/` — backend adapters (Hermes, Pi, Codex, Stokowski, future Claude/OpenAgent/OpenClaw)
- `bridges/` — MCP or transport bridges exposing workers in normalized form
- `control-planes/` — host-specific calling adapters for Pi/Hermes/OpenClaw/TUI
- `external-services/` — clients for external systems such as `ariadne-fact`

This directory is documentation-first for now.
